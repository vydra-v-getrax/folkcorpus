Не для меня придет весна,
Зеленый луг там разовьется,
А сердце радостно забьется
С восторгом чувств не для меня.
Не для меня взойдет луна,
Осветит рощу и долину,
А соловей весну встречает
Он будет петь не для меня.
Не для меня в большом саду
Распустит роза цвет приметный,
Цветок сорвёшь,
А он не сохнет!
Такая жизнь не для меня.
Не для меня течет река,
Течет с алмазными струями,
А дева с черными бровями,
Она растет не для меня.
Не для меня пасха придет
— Христос воскрес, — из уст прольется.
Такая жизнь не для меня.
А для меня пройдет война,
Умчусь на дальние востоки,
Где нерпа ждет давно меня.
